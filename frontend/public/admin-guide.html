📋 Platform Overview

Abbaa Carraa is Ethiopia's first community-driven prize platform where users can participate in prize pools, create pools as Agents/Vendors/Organizations, and earn commissions.

🎯 Mission

Democratize access to valuable prizes by allowing communities to pool small contributions for a chance to win big.

💚 Charity

2% of every contribution supports kidney and heart disease patients in Ethiopia.

💰 Commission

Admin: 20% | Agents/Orgs/Vendors: 10% | Platform: 10%

📌 Quick Facts: Winner gets 100% of target amount. Commissions are added on top, not deducted from winner.

🚀 Getting Started as Admin

1

👥 User Management

User Roles

Individual

Participates in pools. Cannot create pools or earn commissions.

Agent

Can create pools, earn 10% commission. Can recruit users.

Vendor

Lists products, creates pools, earns 10% commission on sales.

Organization

Creates private pools for members, earns 10% commission.

Admin

Full platform control. Personal pools earn 20% commission.

Managing Users

1

Go to Users Tab to see all registered users

2

Change User Role: Use the dropdown menu to assign a new role

3

Ban/Unban User: Click Ban button to restrict access, Unban to restore

⚠️ Caution: Banning a user prevents them from logging in and participating. Banned users cannot create pools or contribute.

🏦 Payment Verification (Bank Transfers)

How It Works

When users pay via Bank Transfer or Telebirr, they upload a payment screenshot. As Admin, you need to verify these payments.

1

Go to Bank Transfers Tab to see pending verifications

2

Review each transfer:

  • User name and contact info
  • Pool name and selected seats
  • Amount paid
  • Reference number
  • Payment screenshot proof
3

Click "View Proof" to see uploaded screenshot

4

Choose action:

  • ✅ Approve & Confirm Seats - Payment verified, seats confirmed
  • ❌ Reject - Payment invalid, seats released

⚖️ Dispute Resolution

Common Dispute Types

  • Winner not paid - Pool creator hasn't delivered prize
  • Payment not verified - User claims they paid but not confirmed
  • Wrong winner selected - Disagreement with draw result
  • Product not as described - Prize quality issues

Resolving Disputes

1

Go to Disputes Tab to see open disputes

2

Exporting Data

Currently, data can be viewed directly in the dashboard. For custom reports, run SQL queries in Supabase.

-- Example: Get all completed contributions
SELECT * FROM contributions WHERE status = 'completed';

-- Example: Get total revenue by month
SELECT DATE_TRUNC('month', created_at) as month, SUM(amount)
FROM contributions
GROUP BY month ORDER BY month DESC;

❓ Frequently Asked Questions

Q

How do I become an admin?

An existing admin must update your role in the database or use the Users tab to change your role to "Admin".

Q

How are winners selected?

When a pool reaches its target, an admin runs the draw using the Draw Management page. A cryptographically secure random number generator selects the winner.

Q

What if a pool doesn't reach its target?

Pools remain active until the end date. If the target is not reached, participants can request refunds (subject to platform policy).

Q

How long do bank transfer verifications take?

Admins should verify within 1 hour during business hours. Users are notified immediately upon verification.

Q

Can I edit a pool after it's created?

Currently, pools cannot be edited after creation. You can pause or delete problematic pools.

Q

How do I send announcements?

Click the Announce quick action or go to Settings Tab → Create Announcement. Choose target audience and publish.
